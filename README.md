# EM Planner in Carsim

### Control

Longitudinal Control based on PID

Lateral Control based on LQR

### Planning

Path Planning based on S-L graph

Speed Planning based on S-T graph

### Requirements

- CarSim 2019.1

- PreScan 8.5.0

- Matlab 2020a

- Visual Studio 2019 (Optional: for C++ compiler)
